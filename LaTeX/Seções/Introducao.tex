O projeto tem como objetivo apresentar uma proposta de desenvolvimento de um Sistema de Monitoramento e Controle Ambiental. O sistema tem como objetivo coletar e monitorar dados de temperatura e umidade em tempo real, garantindo a precisão das informações e a eficiência no acompanhamento das condições do ambiente.

O sistema operará através de sensores integrados que realizam a leitura constante da temperatura e umidade. Os usuários poderão alternar entre diferentes modos de operação (ligado, desligado ou automático), com o auxílio de indicadores visuais em LED que refletem o estado atual do hardware (Vermelho para Desligado, Verde para Ligado, Amarelo para Automático). Além disso, a solução prevê a emissão de alertas e notificações via e-mail caso os limites de segurança sejam ultrapassados.

A interface web possibilitará a visualização dos dados em tempo real e o gerenciamento de comandos, permitindo uma interação intuitiva com o hardware. O sistema é projetado para oferecer alta confiabilidade nas leituras, baixo consumo de energia e uma experiência de uso fluida e agradável para o usuário final.
