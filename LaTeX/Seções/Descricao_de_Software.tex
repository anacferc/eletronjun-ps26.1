\subsection{Especificações}
O software do sistema foi dividido dessa maneira: \par

*Interface do Usuário (UI):* interação do usuário com o sistema, permite ser ligado, desligado e apresenta as informações climáticas sobre o local em que se encontra. \par
*Controle de Dados:* monitora a passagem dos dados e os direciona para seus respectivos lugares. \par
*Comunicação:* monitora a troca de informações entre hardware e software, e a comunicação de softwares externos como APIs. \par
*Armazenamento de Dados:* gerencia e perpetua o armaazenamento de dados provenientes do local em que se encontra o sensor. \par
